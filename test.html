Basic Computer Concepts
Computers work using a binary system, which includes only two digits: 0 and 1. These digits are called "bits," and eight bits make up one byte. All data inside a computer‚Äîtext, images, sounds‚Äîis stored and processed using these binary codes. The main component of a computer is the CPU (Central Processing Unit), which acts like the brain of the system, handling instructions and calculations. The motherboard connects all other components and allows them to communicate. RAM (Random Access Memory) temporarily holds data for fast access while the system is running. The hard drive is the permanent storage device that saves programs and files. The power supply unit provides energy to all parts of the computer. Input devices like the keyboard and mouse let users send data into the system. Output devices like monitors and printers show the results of what the computer does.
